# Powerbi-sales-dashboard
Interactive Power BI dashboard analyzing sales performance with KPIs, trends, and dynamic insights.

## Project Overview
This project presents an interactive Power BI dashboard designed to analyze sales performance and business trends. It provides clear insights into revenue, product performance, and regional sales.

## Key Features
- KPI metrics: Total Sales, Quantity, and Growth
- Sales trend analysis (monthly & yearly)
- Top-performing products and regions
- Dynamic filtering using slicers (Date, Product, Region)
- User-friendly and interactive design

## Insights Generated
- Identified high-performing products contributing to revenue
- Analyzed sales trends over time
- Compared regional performance for better decision-making

## Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation

## Project Highlights

- Developed an interactive Power BI sales dashboard with multiple visualizations
- Used different chart types (bar, pie, and trend visuals) to represent sales data clearly
- Displayed sales distribution by product, region, and category
- Built visual comparisons to identify top-performing segments
- Organized dashboard with a clean and structured layout for easy understanding
- Enabled data exploration using filters/slicers
- Focused on clear data presentation for business insights
